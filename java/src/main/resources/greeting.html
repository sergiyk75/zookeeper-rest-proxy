Zookeeper REST Proxy Service (%s)

Supported commands

  • Tree - List node's child tree: curl -XGET http://127.0.0.1:8889/v1/tree/one
  • List - List node's children: curl -XGET http://127.0.0.1:8889/v1/list/one
  • Get - Get node data: curl -XGET http://127.0.0.1:8889/v1/get/one/two/three
  • Set - Set node data. : curl -XPUT http://127.0.0.1:8889/v1/set/one/two/three -d 'some-data'
  • Delete - Delete node: curl -XDELETE http://127.0.0.1:8889/v1/delete/one
